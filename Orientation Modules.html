List of Orientation Modules

Module 1 - General Library Orientation This module presents about the university library, it's collection and services and what the user should note when borrowing and staying inside.


Module 2 - Access and Location This module teach on how you as our users can utilize print and non-print materials within the university library with the OPAC and other library search tools.


Module 3 - Electronic and Oline Resources This module focuses on the electornic databases and major online resources subscription of the university library - Digital Newspaper on DVD, EBSCO Host, Britannica Academic Edition and Access Science.


Module 4 - Citation Styles This module will guide our users on how to properly cite a resource using the two famous citations style - APA Style and MLA Style.


Module 5 - TURNITIN This module helps our user to avoid commitiing plaigiarism by utilizing a similarity work checker - TURNITIN
